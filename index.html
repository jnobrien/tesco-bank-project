Tesco Treasure Quest

Tesco Bank Inspired Treasure Quest: Financial Freedom

Money doesn't grow on trees...but that treasure chest sure looks like it might be full of gold! Only problem? It's locked. Use the two financial puzzles to determine the code to unlock the treasure chest and your financial freedom.


This game was created for a Code First Girls JavaScript Kickstarter course sponsored by Tesco Bank and uses Tesco Bank articles to inform the puzzles. However, this game is not affiliated with Tesco Bank.


The treasure appears to be locked with a combination consisting of 5 letters followed by 6 numbers (no spaces).
